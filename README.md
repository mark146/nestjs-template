# nestjs 템플릿

이 프로젝트는 nestjs를 사용한 개발을 위한 기본 폴더 구조를 제공합니다. 설계에 대한 지속적인 학습을 통해 더 나은 구조로 지속적으로 업데이트될 예정입니다.


## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
